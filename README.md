## How to build & run
In order to run application, you only need JDK and Maven installed.
```
$ cd coop-pathfinder
$ mvn spring-boot:run
```

## Used technologies

* JavaFX
* Spring Framework
* Spring Boot
* Spring Boot JavaFx Support
